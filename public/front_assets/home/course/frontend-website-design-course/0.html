Description

This course content includes the foundation of website design and the extensive teaching of HTML, CSS & JavaScript, the three languages used for frontend design. HTML stands for Hypertext Markup Language and it is responsible for the structuring of the website. CSS stands for Cascading Stylesheet and it is responsible for the styling of the website. JavaScript is the main programming language of the browser and it is used to make elements interactive and also to add basic functionality to our website.
